# Yggdrasil
Habit building website for individuals with ADHD

##Project Setup
- Open a terminal where the project is to be placed
- Check node.js and npm are present by typing "node --version" then "npm --v"
- Type in "npx create-react-app yggdrasil-app"
- cd into "yggdrasil-app" and type in "npm start"
- Log into the firebase console and follow the instructions to create a new project
- Inside the project directory, type in "npm install firebase"
-copy the firebase config from the console after linking it to the app name
- create a firebase.js setup file in the src
- Edit the App.js file to test the webpage manipulation
- Web app deployed
